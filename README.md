SEKOCI - Landing Page
Website Landing Page untuk SEKOCI, sebuah platform digital inovatif yang dirancang untuk mendukung administrasi, pengelolaan, dan pengembangan pendidikan (Sistem Informasi Sekolah) di Indonesia.

Fitur Utama
- Modern & Clean UI/UX: Desain antarmuka yang profesional, elegan, dan user-friendly sesuai standar platform EduTech (SaaS).
- Fully Responsive Design: Tampilan web otomatis menyesuaikan dengan sempurna di berbagai perangkat (Desktop, Tablet, dan Mobile) menggunakan implementasi *CSS Grid* & *Flexbox*.
- Smooth Animations: Dilengkapi dengan efek fade-up & pop-out menggunakan kombinasi CSS Transitions dan fitur `IntersectionObserver` di JavaScript yang ringan dan mulus.
- Infinite Marquee Slider: Menampilkan daftar klien dan mitra secara otomatis dan looping mulus tanpa putus.

Teknologi yang Digunakan
Project ini dibangun murni menggunakan Native Web Technologies tanpa framework tambahan agar performa maksimal dan ringan:
- HTML5 (Semantik struktur)
- CSS3 (Styling, Animasi, Flexbox, Grid)
- Vanilla JavaScript (Logic interaksi, IntersectionObserver API, Web Animations API)
